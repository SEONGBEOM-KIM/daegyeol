# M5 사전 성향 질문 테스트 체크리스트

## 자동 검증

```bash
npm run test:m5
npm run build
```

## 브라우저 검증

1. 로그인한 뒤 `/account`에서 `성향 질문 시작하기`를 누른다.
2. `/preferences`에 공통 성향 질문 5개가 표시되는지 확인한다.
3. 답변을 빠뜨린 채 저장할 수 없는지 확인한다.
4. 모든 질문에 답하고 `성향 저장하기`를 누른다.
5. `/account`에서 저장 완료 메시지와 `성향 수정하기`가 보이는지 확인한다.
6. 다시 `/preferences`에 접속해 이전 답변이 선택되어 있는지 확인한다.
7. Supabase `profiles`에서 해당 사용자의 `preference_summary`와
   `onboarding_completed = true`를 확인한다.

## 완료 기준

- 로그인한 사용자가 질문 5개에 답할 수 있다.
- 답변이 자신의 `profiles` 행에만 저장된다.
- 다시 접속했을 때 이전 답변이 유지된다.
- 비로그인 사용자는 성향 질문 화면에 접근할 수 없다.
