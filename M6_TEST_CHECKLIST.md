# M6 고민·선택지 입력 테스트 체크리스트

## 자동 검증

```bash
npm run test:m6
npm run build
```

## 브라우저 검증

1. 로그인 후 `/account`에서 `+ 새 고민`을 누른다.
2. 고민 제목에 `점심 뭐 먹지?`를 입력한다.
3. 선택지에 `김치찌개`, `돈가스`를 입력한다.
4. 선택지를 추가하고 삭제할 수 있는지 확인한다.
5. `고민 저장하기`를 누른다.
6. `/account` 최근 고민에 제목이 표시되는지 확인한다.
7. Supabase에서 `decisions` 1행과 연결된 `options` 2행을 확인한다.
8. 선택지를 2개 미만으로 제출하면 저장이 거부되는지 확인한다.

## 완료 기준

- 고민 제목과 최소 2개 선택지를 입력할 수 있다.
- 고민과 선택지가 로그인 사용자 ID로 저장된다.
- 선택지가 2개 미만이면 서버에서도 저장되지 않는다.
- 다른 사용자의 고민과 선택지는 RLS로 조회할 수 없다.
