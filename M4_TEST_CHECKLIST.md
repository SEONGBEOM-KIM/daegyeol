# M4 로그인·로그아웃 테스트 체크리스트

## 자동 검증

```bash
npm run test:m4
npm run build
```

## 브라우저 검증

1. `npm run dev`를 실행한다.
2. `/login`에서 가입한 이메일과 올바른 비밀번호로 로그인한다.
3. `/account`로 이동하고 로그인 이메일이 표시되는지 확인한다.
4. 로그아웃 버튼을 누르고 `/login`으로 이동하는지 확인한다.
5. 로그아웃 상태에서 주소창에 `/account`를 입력한다.
6. 로그인 안내 오류와 함께 `/login`으로 이동하는지 확인한다.
7. 잘못된 비밀번호로 로그인하고 오류 메시지가 표시되는지 확인한다.

## 완료 기준

- 올바른 계정으로 로그인된다.
- 잘못된 비밀번호가 거부된다.
- 비로그인 사용자는 `/account`를 볼 수 없다.
- 로그아웃 후 보호 화면 접근이 차단된다.
